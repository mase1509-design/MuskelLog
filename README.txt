Trainings Log - Lokale Android Web App

Anleitung:

1. Alle Dateien auf dein Android Handy kopieren.
2. index.html im Browser (Chrome) öffnen.
3. Über das Menü 'Zum Startbildschirm hinzufügen' auswählen.
4. Die App wird mit dem Icon installiert und läuft komplett offline.
5. Alle Daten werden im LocalStorage deines Geräts gespeichert.

Viel Erfolg beim Training!
